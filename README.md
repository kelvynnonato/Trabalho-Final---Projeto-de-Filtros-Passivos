# Projeto de Filtros Passivos
## Kelvyn Augusto Waltrick Nonato

 Você foi contratado(a) como engenheiro(a) para projetar um crossover passivo para uma
 caixa de som de duas vias (woofer e tweeter). O objetivo é garantir que os sinais de baixa
 frequência sejam enviados apenas para o woofer e os de alta frequência apenas para o
 tweeter, com máxima fidelidade e uma transição suave
